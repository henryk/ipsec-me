Step 1: Install the StrongS/WAN app

Install the official StrongS/WAN app from the Google play store.

You can also scan the following QR code to open the Google play store:

Step 2: Import the configuration

Import the configuration profile into the app by opening this link with the device: Import profile

You can also scan the following QR code to open this link:

After you have opened the configuration profile for import you will see the "Import VPN profile" dialog. Tap the "IMPORT VPN CERTIFICATE FROM PROFILE" button. You'll be asked for a name for the certificate, just accept the default, and for permission for StrongS/WAN to actually use the certificate, tap 'ALLOW'. Finally, tap 'IMPORT' in the StrongS/WAN import VPN profile dialog.
